# My First Spark Job

Welcome to your first Spark job! This is a simple file that you can use to test PySpark.

## Introduction

This is a README file that contains some information about the project.

## Features

- Simple text file for testing
- Uses Spark's RDD operations
- Works with `sc.textFile`

## How to Run

1. Use `sc.textFile("README.md")` to load the file.
2. Perform basic operations like `count()` to get the number of lines.
3. Enjoy the power of Spark!

Good luck, and have fun with Spark!
